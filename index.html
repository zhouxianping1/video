外贸三语 WordPress 网站｜合规方案摘要（1页）

定位：外贸机械行业 · 中文/英文/越南语 · SEO 导向 · 可长期运营（用于客户/老板快速确认）

一、网站结构（必交付）

  • Home / About Us / Products
  • 产品分类：Press Brake / Shearing / Laser / Automation / Custom Solutions…
  • 产品详情页 / News / Case / Contact
  • Inquiry Form（询盘表单）/ 全站多语言切换

二、多语言系统（合规重点）

  • 使用 WPML 或 Polylang Pro（正版）
  • URL 语言路径：/en、/vn
  • 页面/分类/产品：可独立翻译
  • 多语言 sitemap + 多语言 SEO（Title/Description 独立）
  • 任意页面可切换语言

三、内容建设（不只是空壳）

  • 首页完整布局 + Banner 设计
  • 3–5 个分类页（每类示范 ≥3 个产品）
  • About Us / Contact（Google Map + Form）
  • 询盘表单提交到指定邮箱
  • WhatsApp / 电话 / 邮箱快捷按钮

四、产品详情页（B2B 产品系统）

  • 图片 / 参数（结构化）/ 视频 / PDF 下载
  • 询价按钮（表单/WhatsApp）
  • 支持 SEO：Schema Product
  • 可扩展：后期新增产品与分类不返工

五、SEO 技术要求（Google 基础合规）

  • 独立 Title / Meta Description（多语言）
  • URL 静态化、H1/H2、Breadcrumb（面包屑）
  • robots.txt、sitemap.xml（含多语言）
  • Schema：Organization + Product
  • 图片 ALT、缓存与图片压缩优化
  • 速度目标：PageSpeed 移动 ≥60、桌面 ≥80
SEO 插件：RankMath 或 Yoast（正版） 缓存：WP Super Cache（配置完成） 图片：Smush（压缩与优化）

六、必装插件（必须正版）

  • Elementor Pro
  • WPML / Polylang Pro
  • RankMath / Yoast SEO
  • WP Super Cache / Smush
  • Contact Form 7 / Google Site Kit

七、Google & 安全性能

  • GA4 + Search Console（可选 GTM）
  • SSL（HTTPS）/ 防垃圾邮件
  • 自动备份（每日/每周）

八、网站所有权（强制）

  • 域名、主机后台、WordPress 超级管理员权限
  • FTP/文件管理器（全源码）与 phpMyAdmin 数据库权限
  • 主题/插件均为正版，并提供授权方式/激活码（可迁移）
结论：本方案为“可做 Google 获客”的完整合规方案，包含多语言 SEO、产品系统与正版授权；不等同于低价“展示型网站方案”。

九、周期与预算（参考区间）

  • 周期：25个左右 工作日
  • 合理预算：40,000,000 – 45,000,000 VND（视内容量、视觉设计与资料齐全程度浮动）
文档用途：客户确认 / 老板审批 （标准摘要）
